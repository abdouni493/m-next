## 🗄️ DATABASE STRUCTURE

### Table: caisse_transactions
```
CREATE TABLE caisse_transactions (
  id UUID PRIMARY KEY,
  transaction_type TEXT (encaissement|decaissement),
  amount DECIMAL(12,2),
  description TEXT,
  transaction_date TIMESTAMP,
  created_at TIMESTAMP,
  updated_at TIMESTAMP,
  created_by UUID,
  is_active BOOLEAN
);
```

### Indexes
```
- idx_caisse_type: Quick filter by transaction type
- idx_caisse_date: Fast date sorting
- idx_caisse_active: Filter active transactions
```

### View: caisse_summary
```
Calculates:
- total_encaissements: SUM of all deposits
- total_decaissements: SUM of all withdrawals
- balance: Deposits - Withdrawals
- total_transactions: COUNT of transactions
```
