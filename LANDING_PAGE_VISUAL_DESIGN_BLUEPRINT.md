## 🎨 Color Scheme

### Hero Section
```
Background Image: User-uploaded (with overlay)
Overlay Gradient:  Black 70% → 50% → 80% (varying opacity)
Text Colors:
  - Title:         Cyan → Blue → Purple (gradient)
  - Slogan:        Pink (#ec4899)
  - Description:   Light Gray (#d1d5db)

Animated Elements:
  - Shape 1:       Cyan glow (rotating)
  - Shape 2:       Purple glow (rotating)
```

### Offers Section
```
Background:      Red/Black gradient
Border:          Red 500/50 opacity
Title Text:      Red → Pink gradient
Price Old:       Gray strikethrough
Price New:       Red bold
Button:          Red → Pink gradient (hover: darker)
Hover Effect:    Y-axis move up 10px, shadow increase
```

### Products Section
```
Background:      Black gradient
Border:          Cyan 500/30 opacity
Title Text:      Cyan → Blue gradient
Price:           Cyan bold
Button:          Cyan → Blue gradient
Hover Effect:    Y-axis move up 10px, shadow increase
Image Zoom:      1.1x scale on hover
```

### Footer
```
Background:      Black 80% with border top
Text:            Gray 400 (muted)
Text-Sm:         Gray 400 smaller size
```

## ✨ Animation Specifications

### Page Load Sequence
```
Timeline:
0.0s:    Hero section fades in
0.1s:    Logo animates in
0.2s:    Title animates in
0.3s:    Slogan animates in
0.4s:    Description animates in
0.5s:    Buttons animate in
0.6s:    Scroll indicator starts animating

Easing:  ease-out (smooth deceleration)
```

### Background Image Animation
```
Initial State:   scale(1.1) + opacity(0)
Final State:     scale(1) + opacity(0.6)
Duration:        1.2 seconds
Easing:          cubic-bezier(0.34, 1.56, 0.64, 1)
```

### Animated Shapes
```
Shape 1 (Top-Right):
  Rotation: 360° continuous
  Duration: 20 seconds
  Easing:   linear
  Color:    Cyan glow
  Size:     w-96 h-96
  Blur:     blur-3xl
  Opacity:  20%

Shape 2 (Bottom-Left):
  Rotation: -360° continuous
  Duration: 25 seconds
  Easing:   linear
  Color:    Purple glow
  Size:     w-96 h-96
  Blur:     blur-3xl
  Opacity:  20%
```

### Scroll Indicator
```
Y-axis Animation: [0, 10, 0] pixels
Duration:         2 seconds
Repeat:           Infinite
Easing:           Linear
Direction:        Up and down
```

### Button Hover
```
Scale:       1.02x
Y-movement:  -2px (up)
Shadow:      Enhanced
Duration:    0.2 seconds
Transition:  Smooth
```

### Card Hover (Products/Offers)
```
Y-movement:  -10px (up)
Duration:    0.3 seconds
Shadow:      Color-specific glow
Transition:  Smooth
```

### Image Hover (Products)
```
Scale:       1.1x
Duration:    0.3 seconds
Overlay:     Fade in black 60%
Transition:  Smooth
```

## 📱 Responsive Breakpoints

### Mobile (< 640px)
```
Hero Section:
  - Padding:     16px (sm:px-6 gives px-4)
  - Font size:   5xl (instead of 6xl)
  - Button:      Stack vertically
  - Logo:        h-20 (instead of h-24)

Offers/Products:
  - Grid:        grid-cols-1 (single column)
  - Spacing:     gap-6 (reduced)
  - Card:        Full width

Footer:
  - Font:        Text-sm
  - Padding:     12px
```

### Tablet (640px - 1024px)
```
Hero Section:
  - Font size:   5xl-6xl
  - Button:      Side by side with flex-wrap
  - Padding:     lg:px-8

Offers/Products:
  - Grid:        grid-cols-2 (md:grid-cols-2)
  - Spacing:     gap-8
  - Cards:       Balanced 2-column

Footer:
  - Standard layout
```

### Desktop (> 1024px)
```
Hero Section:
  - Font size:   6xl-7xl full
  - Button:      Side by side, spaced
  - Padding:     lg:px-8

Offers/Products:
  - Grid:        grid-cols-3 (lg:grid-cols-3)
  - Spacing:     gap-8
  - Cards:       Full 3-column layout
  - Max width:   max-w-7xl centered

Footer:
  - Full width centered
```

## 🎬 Animation Timeline (Detailed)

```
Page Load:
0ms:    HTML renders, CSS applies
100ms:  Images start loading
200ms:  Hero section fade-in starts (0→1 opacity)
300ms:  Background image scale animation (1.1→1)
400ms:  Logo fade and scale (0→1)
500ms:  Title slides up (y:20→0)
600ms:  Slogan slides up (y:20→0)
700ms:  Description slides up (y:20→0)
800ms:  Buttons slide up (y:20→0)
900ms:  Scroll indicator begins loop
1000ms: Page fully loaded and interactive

Scroll Indicator Loop (2s cycle):
0ms:    Y = 0
500ms:  Y = 10px (moving down)
1000ms: Y = 0
1500ms: Y = 10px (moving down)
2000ms: Reset and loop

Rotating Shapes (continuous):
Shape 1: rotate(0°) → rotate(360°) over 20s
Shape 2: rotate(0°) → rotate(-360°) over 25s

Background Gradient Animation (15s cycle):
0%:    background-position: 0% 0%
50%:   background-position: 100% 100%
100%:  background-position: 0% 0%
```

## 🎨 Visual Hierarchy

### Size Order (Large to Small)
```
1. Hero Section Title (6xl-7xl)
2. Offers/Products Titles (5xl)
3. Slogan (2xl-3xl)
4. Description (lg-xl)
5. Subtitles (xl)
6. Product Names (lg)
7. Prices (lg-2xl)
8. Footer Text (sm-base)
```

### Importance Order (Visual Weight)
```
1. Hero Background Image (dominates page)
2. Store Name (largest text)
3. Logo (center, prominent)
4. CTA Buttons (bright colors)
5. Offers Section (red/pink = attention)
6. Products Section (more products)
7. Footer (minimal)
```

### Color Importance
```
1. Cyan/Blue/Purple (main brand colors)
2. Red/Pink (calls to action, discounts)
3. White (text, contrast)
4. Gray (secondary text)
5. Black (backgrounds, overlays)
```
