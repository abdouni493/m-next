# 🚀 Wilaya Pricing - Quick Start Guide

## 5-Minute Setup

### What Is This?
Set **different delivery prices for each Algerian wilaya** (province) instead of one fixed price for everyone.

**Example:**
- Customers in **Algiers**: 200 DZD (home), 150 DZD (office)
- Customers in **Oran**: 350 DZD (home), 280 DZD (office)

---

## ⚡ Quick Steps

### 1️⃣ Open Settings
```
Website Manager → Delivery Management
```

### 2️⃣ Click Manage Prices
Click the **💰** button on any delivery agency card

### 3️⃣ Add a Wilaya Price
```
1. Pick a wilaya (e.g., "Algiers")
2. Enter home delivery price (e.g., 200.00)
3. Enter office delivery price (e.g., 150.00)
4. Click "Add"
```

### 4️⃣ Done! ✅
Customers in that wilaya now see the correct price automatically.

## 💡 Pro Tips

### Tip 1: Use Agency Defaults for Quick Start
- If you don't set a wilaya price → Uses agency default
- Good for starting out
- Can add specific prices later

### Tip 2: Copy Prices from Similar Regions
- **Nearby wilayas** often have similar distances
- Use same price to save time
- Update later if needed

### Tip 3: Factor in Distance
- **Close wilayas**: Lower prices
- **Far wilayas**: Higher prices
- **Mountain areas**: Higher prices (more difficult terrain)

### Tip 4: Consider Season
- **Summer**: Roads good → Lower prices possible
- **Winter**: Roads difficult → Higher prices justified

## ❓ FAQ

### Q: What if I don't add wilaya prices?
**A:** Customers automatically get the agency's default price. Works perfectly fine!

### Q: Can I change prices anytime?
**A:** Yes! Click ✏️ to edit or 🗑️ to delete. Changes apply instantly.

### Q: How many wilayas can I set?
**A:** All 58 Algerian wilayas are supported!

### Q: What's "Default Price"?
**A:** The base price you set when creating the agency. Used if no wilaya-specific price exists.

### Q: Do I need this feature?
**A:** No! It's optional. Use if you have different distances or costs by region.

## 🎬 Step-by-Step Demo

### Adding Your First Wilaya Price (2 minutes)

```
STEP 1: Open Website Manager
└─ Look for button in top menu

STEP 2: Go to Delivery Management
└─ Click the "Delivery Management" tab

STEP 3: Find Your Agency
└─ Look for delivery agency card
   Example: "Yassir Livraison"

STEP 4: Click Manage Prices
└─ Click the 💰 button
   (Should be first button on card)

STEP 5: Select Wilaya
└─ Click the dropdown: "Select Wilaya"
└─ Type or scroll to find: "Algiers"
└─ Click to select

STEP 6: Enter Prices
└─ Home price: 200.00
└─ Office price: 150.00

STEP 7: Save
└─ Click the "Add" button

STEP 8: Verify
└─ Should see "Algiers" in list below
└─ Should show "200.00 | 150.00"

DONE! ✅
```

## 💬 Need Help?

### If prices don't update:
1. Refresh the page
2. Check spelling of wilaya name
3. Make sure you clicked "Add" or "Update"
4. Check browser console for errors

### If you can't find the button:
1. Make sure you're on "Delivery Management" tab
2. Make sure you're looking at agency cards
3. Try scrolling down
4. Try zooming out (Ctrl + -)

### If wilaya list is empty:
1. Try typing in the field
2. Try scrolling in the dropdown
3. Check if dropdown is actually open
4. Refresh the page
