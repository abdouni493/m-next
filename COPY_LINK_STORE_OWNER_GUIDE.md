# How to Use Copy Link Feature - Store Owner Guide

## 🎯 Quick Overview

With the new Copy Link feature, you can easily share product links that automatically take customers directly to checkout with the product already selected!

---

## 📋 Step-by-Step: How to Copy a Product Link

### Step 1: Login to Website Manager
```
Open: https://store.com/admin
→ Enter credentials
→ Login successful
```

### Step 2: Navigate to Products Section
```
Dashboard
├─ 📦 Inventory
├─ 💰 Offers  ← CLICK HERE
├─ ⭐ Special Offers  ← OR HERE
├─ 📦 Packages
└─ More...
```

### Step 3: Find Your Product
```
In the Offers/Special Offers tab, you'll see product cards:

┌─────────────────────────────┐
│ [Product Image]             │
│ Solar Charger 50W           │
│ Brand: XYZ Electronics      │
│ Original Price: 3000 DZD    │
│ Offer Price: 2500 DZD (17%) │
│                             │
│ [👁️] [📋] [×]  ← Click 📋  │
└─────────────────────────────┘
```

### Step 4: Click "Copy Link" Button
```
Hover over product card
↓
You'll see 3 action buttons:
[👁️]  = View/Hide product
[📋] = Copy Link ← CLICK THIS
[×]  = Delete product

Click [📋] Copy Link button
```

### Step 5: See Confirmation
```
✅ Toast Notification:
   "📋 Copié"
   "Lien copié - Le produit sera sélectionné automatiquement"
   
This means:
✓ Link is copied to clipboard
✓ Product will auto-select when opened
✓ Ready to share!
```

## 📱 How to Share the Link

### Method 1: WhatsApp (Most Popular)

```
1. Copy link (see above steps 1-5)
2. Open WhatsApp

4. Long-press in message box
```

## 🎯 Best Practices

### ✅ DO:

1. **Use Descriptive Messages**
   ```
   "Hi! Here's the charger I mentioned 👇"
   https://store.com/order?product_id=xyz
   ```

2. **Add Urgency When Appropriate**
   ```
   "Limited offer ends today! Get it now:
   https://store.com/order?product_id=xyz"
   ```

3. **Share Multiple Products**
   ```
   "Option 1: Solar Charger
   https://store.com/order?product_id=abc
   
   Option 2: Power Bank
   https://store.com/order?product_id=def
   
   Which one do you prefer?"
   ```

4. **Use on Social Media**
   ```
   Posts → More visibility
   Stories → More urgency
   Comments → Direct responses
   ```

5. **Include Product Details**
   ```
   "Solar Charger 50W - 2500 DZD (Was 3000!)
   Includes: USB ports, weatherproof case
   https://store.com/order?product_id=xyz"
   ```

---

### ❌ DON'T:

1. **Don't spam** - Share links appropriately, not excessively
2. **Don't hide the link** - Make it visible and easy to click
3. **Don't modify the link** - Keep it exactly as copied
4. **Don't share wrong product** - Double-check before sending
5. **Don't forget context** - Explain why you're sharing the product

## ❓ FAQ

**Q: What if customer doesn't have the product in cart?**
A: The product automatically adds to cart when link is opened!

**Q: Can customer change quantity?**
A: Yes! After link opens, they can adjust quantity before checkout.

**Q: Does the link expire?**
A: No, the link never expires. It's permanent.

**Q: What if product is out of stock?**
A: Link still works, but customer will see "Out of Stock" notice.

**Q: Can I track who clicked my link?**
A: Not yet, but this feature is coming soon!

**Q: Does the link work on WhatsApp Business?**
A: Yes! Works on all platforms.

**Q: What if I share same link to 10 people?**
A: All 10 can use it independently. No limits.

**Q: Can customer use link multiple times?**
A: Yes! Every time they open it, product gets added to a new cart.

**Q: Does it work on mobile?**
A: Yes! Perfectly optimized for mobile.
